How to run React App

The URL below are a set of instructions on how to create a React App with Vite:
https://www.freecodecamp.org/news/how-to-install-tailwindcss-in-react/

This React App uses Vite which will require a Node.js version 22+.  Below is the URL of instructions on how to update your Node to a later version if using Homebrew does not work.
https://www.geeksforgeeks.org/node-js/update-node-js-and-npm-to-latest-version/

When coming back to run the React App, the following commands must be entered into the terminal:
npm install
npm run dev

After those commands are entered, the webpage should be running locally.
